## Catálogo de Servicios

### Aplicaciones Web para Monitoreo Ambiental en Tiempo Real

**Problema que resuelve:** La falta de visibilidad en tiempo real sobre condiciones críticas de cultivos (temperatura, humedad) genera pérdidas por reacción tardía.

**Alcance del servicio:**

- Diseño y desarrollo de plataformas web con visualización en tiempo real
- Integración de sensores IoT (temperatura, humedad, pH, luminosidad)
- Base de datos para historial de variables ambientales
- Dashboard interactivo accesible desde cualquier dispositivo
- Documentación técnica del sistema

**Respaldo técnico:** Proyecto en desarrollo — Plataforma web para monitoreo de cultivos hidropónicos con sensores IoT y dashboard de visualización en tiempo real.

---

### Automatización de Riego Agrícola con IoT

**Problema que resuelve:** Los procesos de riego manual generan desperdicio de agua, ineficiencia operativa y dependencia del factor humano.

**Alcance del servicio:**

- Diseño e implementación de sistema inteligente de riego automatizado
- Sensores de humedad del suelo y actuadores con control remoto
- Alertas configurables por umbrales críticos en tiempo real
- Panel de control web para gestión remota
- Documentación y capacitación básica del sistema

**Respaldo técnico:** Proyecto entregado (2024–2025) — Sistema de automatización de riego para empresa agrícola con sensores de humedad, control remoto y alertas en tiempo real.

**Sector:** AgroTech | **Estado:** Entregado
